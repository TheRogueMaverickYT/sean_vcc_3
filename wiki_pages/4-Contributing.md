# Contributing

We welcome pull requests! If you have optimized the fuel-to-air ratio map or fixed a bug in the anti-wobble steering algorithm, please submit your changes.

1. Fork the project.
2. Create your feature branch (`git checkout -b feature/AmazingBoost`).
3. Commit your changes (`git commit -m 'Add some AmazingBoost'`).
4. Push to the branch (`git push origin feature/AmazingBoost`).
5. Open a Pull Request.
