# Installation

To get the firmware installed on your board's micro-controller, follow these steps:

1. Clone the repository: `git clone https://github.com/TheRogueMaverickYT/sean_vcc_3.git`
2. Connect your board via USB-C.
3. Run the installation script: `./install_firmware.sh`
4. Pray.

**Note:** Ensure you are in a well-ventilated area before doing the first test spool-up.
